================
smitheme.phoenix
================

Alternate skin for the `Silva`_ Management Interface (SMI) for Silva
2.3. This skin doesn't work for Silva 3.

To use this skin, just activate the extension in your ``service_extension``.


Code repository
===============

You can find the source code for this extension in Git:
https://github.com/silvacms/smitheme.phoenix

.. _Silva: http://silvacms.org
